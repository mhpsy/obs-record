# obs-record

专为 Hyprland 桌面设计的 OBS 录屏增强插件。

为"讲解式录屏"(如帮同学讲解简历修改)提供:

- **缩放跟随鼠标** — 快捷键平滑放大,镜头带死区跟随光标(参照 obs-zoom-to-mouse,基于 Hyprland IPC 实现 Wayland 支持)
- **光标可视化** — 常驻高亮圈、点击波纹、快速淡出的移动轨迹
- **固定标注** — 快捷键在屏幕位置钉编号标记/高亮框,一键清除
- **聚光灯模式** — 光标周围保持明亮,其余画面压暗

> An OBS Studio plugin for Hyprland: smooth zoom-to-mouse, cursor highlighting,
> click ripples, pinned annotations and spotlight mode for tutorial-style
> screen recording on Wayland.

🚧 设计阶段,设计文档见 `docs/superpowers/specs/`。
