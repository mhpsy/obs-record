## 缩放
- [ ] SUPER+Z 平滑放大到 2x(约 0.3s),再按平滑还原

- [ ] 鼠标出死区,画面平滑跟随,无突跳
- [ ] 鼠标顶到屏幕四角,画面贴边、无黑边
- [ ] SUPER+C 依次切换 1.5x / 2x / 3x
- [ ] obs-record-cli zoom set 5 生效;zoom set 0.5 回到 1x

## 光标可视化

- [ ] SUPER+T 开轨迹,快速画圈见渐隐拖尾;再按关闭且立即消失
- [ ] 属性面板关闭"点击波纹"后点击无效果
